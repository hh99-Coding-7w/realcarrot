내용
🎉코로나로 끝나가는 이 시기에 다시금 여행에 대한 설렘이 살아날 수 있도록 자신이 갔던 여행지의 정보와 후기등을 공유하는 사이트 🎉누군가에게는 여행지에 대한 정보가 될 수 있고, 누군가에게는 자신이 갔던 여행지의 즐거움을 한번 더 기억하게 되는 그런 사이트를 만들고 싶어서 제작하게 되었습니다.

😎 여행의 민족을 제작한 자랑스러운 개발자들
👉 BE : 강명지, 박선우, 김성현

👉 FE : 임운철, 정대규, 장세화
